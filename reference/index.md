# Package index

## API Key

- [`setlegiKey()`](https://aberuiz.github.io/legihelpR/reference/setlegiKey.md)
  : Set a legiscan api key
- [`getlegiKey()`](https://aberuiz.github.io/legihelpR/reference/getlegiKey.md)
  : Return legiscan API key

- [`getSessions()`](https://aberuiz.github.io/legihelpR/reference/getSessions.md)

- [`getMasterList()`](https://aberuiz.github.io/legihelpR/reference/getMasterList.md)
  : Return legiscan Master List
- [`getMasterListRaw()`](https://aberuiz.github.io/legihelpR/reference/getMasterListRaw.md)
  : Return legiscan Master List for change detection
- [`getBill()`](https://aberuiz.github.io/legihelpR/reference/getBill.md)
  : Return a specific bill
- [`getText()`](https://aberuiz.github.io/legihelpR/reference/getText.md)
  : Return bill text document
- [`getAmendment()`](https://aberuiz.github.io/legihelpR/reference/getAmendment.md)
  : Return amendment document
- [`getSupplement()`](https://aberuiz.github.io/legihelpR/reference/getSupplement.md)
  : Return supplement document
- [`getRollCall()`](https://aberuiz.github.io/legihelpR/reference/getRollCall.md)
